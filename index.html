Predicting Drug Sensitivity in Cancer Cell Lines

Each cancer is different. In this project we developed initial algorithms to predict drug sensitivity of different cancer cell lines. we developed elastic nets and lasso regression models to predict drug sensitivity for different cancer cell lines to customize treatment for individual patients. Models and visualizations were developed in R. Used Principal Component Analysis for dimensionality reduction as each individual patient data contained 1639 Mutations, 23316 Copy Number and 18988 Gene Expressions. One of the major learnings from this project was in using PCA for dimensionality reduction which beomes crucial in feature selection. ...more
